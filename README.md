# ⚽ Football Players Data Analysis

## 📌 Overview

This project explores and analyzes a football players dataset using Python and data visualization libraries. The analysis provides insights into player attributes, teams, positions, and other statistics through various charts and graphs.

## 🚀 Features

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Player distribution by team
* Position-wise analysis
* Statistical summaries
* Data visualization using different chart types

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## 📂 Dataset

The dataset contains information about football players, including:

* Player Name
* Team
* Position
* Age
* Nationality
* Overall Rating
* Market Value
* Other player attributes

## 📊 Visualizations

The project includes:

* Bar Charts
* Horizontal Bar Charts
* Pie Charts
* Histograms
* Line Charts

## 📈 Sample Analysis

* Number of players in each team
* Distribution of player positions
* Age distribution of players
* Comparison of player ratings

## ▶️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/football-players-data-analysis.git
```

### Install Required Libraries

```bash
pip install pandas matplotlib
```

### Run the Notebook

Open the Jupyter Notebook and execute all cells.
